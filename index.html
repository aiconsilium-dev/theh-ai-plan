CHAPTER 01
프로젝트 개요

프로젝트명

법무법인 더 H 황해 — AI 기반 통합 의사결정 및 실행 시스템

프로젝트 목적

  • 법률 업무 전 과정에 AI를 통합하여 의사결정 속도와 실행력을 극대화
  • 단순 검색/조회가 아닌, 실행 가능한 산출물을 즉시 생성하는 AI 시스템 구축
  • 하자소송 분야에서 데이터 기반 경쟁 우위 확보

대상 조직

성명직위역할
천상현대표변호사총괄 의사결정
성현우국장운영 총괄
이창선변호사하자 기술/소송
김돌리변호사하자 기술/소송

핵심 방향

  • "실행까지 만들어주는 AI" — 분석 결과를 홍보물, 제안서, 실행방법론으로 즉시 전환
  • "회의에 참여하는 AI" — 실시간 논점 정리, 전략 제안, 실행안 생성
  • "데이터 기반으로 설득까지 해주는 AI" — 과거 실적/사례를 기반으로 설득 자료 자동 구성

추진 일정 (개요)

단계내용기간
Phase 1백데이터 구축 + 핵심 AI 개발1~3개월
Phase 2실전 적용 + 피드백 반영4~6개월

레이어 간 연동 구조

  • L1 (회의 AI): 회의 중 실시간 요청 발생 시 L2~L5를 호출하여 즉시 결과 반환

CHAPTER 06
핵심 기능 상세

L1. 실시간 회의 AI (Auto-K)

입력 - 처리 - 출력 플로우

음성/텍스트 입력 → STT 변환 + 화자 분리 → 논점 추출 + 분류 → 전략 제안 + 실행안
  • 입력: 회의 음성 실시간 수집, 텍스트 채팅, 안건 문서
  • 처리: 음성 인식(STT), 핵심 논점 자동 추출, 과거 회의 맥락 참조
  • 출력: 논점 정리 문서, 전략 제안서, 실행 방법론, 홍보물 초안

L2. 데이터 분석 AI

입력 - 처리 - 출력 플로우

조건/키워드 입력 → DB 검색 + 필터링 → 통합 분석 + 비교 → 분석 보고서 생성
  • 입력: 하자유형, 공종, 시공사, 금액 범위, 기간 등 조건
  • 처리: 감정서/판결문/민원 데이터 통합 검색, 교차 분석
  • 출력: 통합 분석 보고서, 로펌별 실적 비교, 시공사별 판결 경향

L3. 사례 매칭 AI

입력 - 처리 - 출력 플로우

단지/시공사/사진 → 유사도 계산 + 비전 AI → 사례+판결+감정 매칭 → 매칭 결과 리포트
  • 입력: 단지명, 시공사명, 하자 사진, 하자 설명
  • 처리: 텍스트 임베딩 유사도, 이미지 분류, 규칙 기반 매칭
  • 출력: 유사 사례 목록 (판결결과, 감정금액, 승소율 포함)

L4. 영업 지원 AI

입력 - 처리 - 출력 플로우

대상 단지/시공사 → 실적 조회 + 사례 수집 → 템플릿 자동 삽입 → 발표 자료 완성
  • 입력: 입찰 대상 단지, 시공사, 설명회 일정
  • 처리: 과거 소송 실적 조회, 유사 사례 수집, 승소율/감정금액 집계
  • 출력: 입찰설명회 발표 자료, 실적 기반 설득 보고서
